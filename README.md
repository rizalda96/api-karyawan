# Tech stack

- Nest Js
- Mysql
- NodeMailer
- Bcyrt

# Penjelasan Arsitektur
## Project Architecture Overview
Project ini menggunakan Nest js, Nodemailer untuk mengirim pesan email, dan juga di lengkapi dengan bcryt untuk hashing password



## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
